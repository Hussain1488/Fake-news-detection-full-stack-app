# Fake News Detection

Short description of what the app does.

## Features

- Reading news 
- Vectorizing Test and Title of news
- Detecting news using pretrained model
- using llm model for reasoning

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

Backend:
- FastAPI
- PostgreSQL


DevOps:
- Docker
- Docker Compose
- CI/CD
- Github Action
